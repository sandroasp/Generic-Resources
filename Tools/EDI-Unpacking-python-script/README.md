# EDI Unpacking Python Script for Notepad++
Add to Python script to ident EDI file
 * Open Notepad++.
 * Go to Plugins → Plugins Admin.
 * Search for PythonScript.
 * Install and restart Notepad++.
 * Name it: edi_unpack.py.
 * And Paste the available here
